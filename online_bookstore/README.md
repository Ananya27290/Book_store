# Online Bookstore Management System

## Description
A web-based Online Bookstore application developed using Flask and MySQL.
The system supports user authentication, book browsing, order management,
and admin functionalities.

## Technologies Used
- Python (Flask)
- MySQL
- HTML, CSS, Bootstrap
- Flask-Login

## Features
- User Registration & Login
- Browse Books
- Add to Cart & Checkout
- Order History
- Admin Book Management
- Secure Authentication

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Configure database in `config.py`
4. Run:
   python app.py
